### 说明：添加综合资讯接口文档
## 1.安装
> http://(host:port)/newsManage/addComplexInformation
>>> 如：http://localhost:3001/newsManage/addComplexInformation
请求格式 `POST`

### 请求头
```
Content-Type=application/json
```
```json
{
          "title" :"都是一些什么标题啊",
          "reTitle":"dsadas",
          "label":"党委会议",
          "content":"党委会议党委会议党委会议党委会议",
}
```
