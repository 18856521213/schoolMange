### 说明：添加校园资讯接口文档
## 1.安装
> http://(host:port)/newsManage/addSchoolInformation
>>> 如：http://localhost:3001/newsManage/addSchoolInformation
请求格式 `POST`

### 请求头
```
Content-Type=application/json
```
```json
{
          "title" :"都是一些什么标题啊",
          "reTitle":"dsadas",
          "content":"<p>校园资讯内容</p>",
}
```
### 返回格式

```json
{
    "status": 200,
    "successMessage": "添加成功",
    "errMessage": null,
    "data": null,
    "success": true,
    "failure": false
}
```
