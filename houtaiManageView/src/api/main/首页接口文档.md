### 说明：查询综合资讯接口文档
## 1.安装
> http://(host:port)/newsManage/findComplexInformation
>>> 如：http://localhost:3001/newsManage/findComplexInformation
请求格式 `POST`

### 请求头
```
Content-Type=application/json
```
```json
{
          "page" :1,
}
```
### 返回格式

```json
{
    "status": 200,
    "successMessage": null,
    "errMessage": null,
    "data": {
        "title":"",
        "titleId":"",
        "reTitle":"",
        "cintent":"",
        "total":12
    },
    "success": true,
    "failure": false
}
```
